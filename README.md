# Dr. Pradeep Kumar — Neurologist Profile Website

A modern, fast, and SEO-optimized professional portfolio website for Dr. Pradeep Kumar, an expert neurologist serving Lucknow and Kanpur. Built with [Next.js](https://nextjs.org/) and React.

## Features

- **Modern Tech Stack**: Next.js App Router, React 19, and TailwindCSS v4.
- **Responsive & Accessible**: Features Radix UI primitives ensuring the highest standard of web accessibility and interactions.
- **Clean Animations**: Framer Motion powers the smooth scroll reveals and layout transitions.
- **Exceptional Typography**: Integrates optimized Google Fonts (Fraunces & DM Sans) utilizing Next.js Font Optimization.
- **Search Engine Optimized**: Embedded standard and JSON-LD structured data for Google Rich Results.

## Getting Started

First, ensure you have your Node.js environment ready. 

Install the dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Making Production Builds

To build the application for deployment, run:

```bash
npm run build
```

You can start the production representation using:

```bash
npm run start
```
